Computer Security: The Risks

In the contemporary era, people are more attached with the technology. The output of technology is computer ,laptops etc. Computer plays an important role and make our life easy especially the people who belongs to the united states. Businessman, different organizations are more familiar with the computers and laptops. As we run our computer for the data storage , entertainment etc but to secure the information in it, we need the computer security. As a result it provide the protection from any unusual activity done by unknown people.


Safety provide to the computer operating system by computer security. Computer security is very necessary to run the computer. It is a feature to install in computer to prevent it from threats and incidents by unauthorized person without user consent. Nowadays different types of computer security which are developed in computer . Antiviruses, firewall, data encryption, password protection are the examples of computer security which prevent from the mischief, information theft, spying, resource theft(UKEssays,2018).


However, there are lots of threats which affect the computer security. Malware software have different forms have forms and causes threats to computer. It is also called the malicious software which steal the private information by sending the mails, links through which the hackers track the information when any person click on these links. On the otherhand, Virus copy the code of program and multiply which affects the speed of computer system. When viruses attack on our system then antivirus program increase the security and stop the virus to enter and interupt the system. It blocks the viruses and prevent our system from hackers. Another threat is torjan in which people sent gift which look like a gift to others. There are more threats like worm, exploit etc that are harmful for the system. (Gaffney Melissa,2019)

They do their work from home,transaction, online shoping through them. It protect our personal and financial information.
